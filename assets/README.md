# Assets

This folder contains diagrams and images used across the cybersecurity notes for illustration and clarification.

## Contents
- Network diagrams
- Security architecture diagrams
- Protocol flow illustrations
- Screenshots (labs and tools)

## Naming Convention
- lowercase
- hyphen-separated
- descriptive filenames  
  e.g. `tcp-three-way-handshake.png`

## Usage
Assets are referenced from Markdown files using relative paths.
